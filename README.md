# 这是一个能自动生成android studio SelectorDrawable文件模板插件

   - **插件使用方法**
      1. 选中要要 drawable 目录 下的xml 文件
      2. 右键单击，选择"Generator Selector Drawable" 菜单项
      3. 在选中的文件目录下生成SelectorDrawable成功。
